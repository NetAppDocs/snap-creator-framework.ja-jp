= クローンの管理
:icons: font
:imagesdir: ../media/


[role="lead"]
クローンの管理では、（ * Actions * 設定を使用して）クローンを作成したり、バックアップコピーをソースとしてマウントしたり、クローンを削除したり、クローンをアンマウントしたりできます。

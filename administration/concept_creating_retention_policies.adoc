= 保持ポリシーを作成します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
（構成ファイルに含まれる）ローカル保持ポリシーの代わりにポリシーオブジェクトを使用する場合は、保持ポリシーを作成できます。

保持ポリシーを作成する際に、バックアップタイプとポリシースケジュールを作成することができます。

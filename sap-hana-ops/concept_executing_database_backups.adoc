= データベースのバックアップを実行中です
:icons: font
:imagesdir: ../media/


[role="lead"]
SAP HANA データベースは、 Snap Creator の GUI またはコマンドラインを使用してバックアップできます。バックアップのスケジュールを設定するには、 GUI でスケジューラを使用するか、 cron などの外部スケジューラとコマンドラインを組み合わせて使用します。

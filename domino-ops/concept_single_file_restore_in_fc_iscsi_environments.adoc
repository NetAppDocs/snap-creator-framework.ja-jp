= SAN 環境で単一ファイルのリストアを実行する
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SAN 環境で単一ファイルのリストアを実行する場合は、 SnapDrive を使用して、 Snap Creator エージェントホストに Snapshot コピーをマウントできます。リストアするファイルを Data Restore Directory にコピーし、 Snap Creator のカスタムアクションを使用してリストア処理を完了できます。
